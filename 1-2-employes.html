Des exos pour s'entraîner

1 - INSERT INTO, requête d'insertion

  • Pour insérer un enregistrement, 1ère méthode : INSERT INTO employes (prenom, nom, sexe, service, date_embauche, salaire) VALUES ('Alexis', 'Richy', 'm', 'informatique', '2011-12-28', 1800);
  • Pour voir l'ID d'Alexis, il y a deux façons : SELECT * FROM employes WHERE prenom='alexis'; ou bien SELECT * FROM employes ORDER BY id_employes DESC LIMIT 1;
  • Pour insérer un enregistrement, 2ème méthode : INSERT INTO employes VALUES (NULL , 'Jeanne', 'Richy','m','informatique','2012-01-28', 1800)
  • NULL nous permet de laisser le champs id_employes se remplir seul car il est auto-incrémenté.

2 - Requête de modification

a) Maj avec UPDATE

  • Augmenter le salaire d'un employé : UPDATE employes SET salaire=1871 WHERE id_employes=990;
  • On utilise toujours des id pour mettre à jour les informations de la base de donnée car c'est le seul élément unique.
  • Pour faire plusieurs mises à jour en même temps il faudra respecter l'ordre des colonnes et les séparer par une virgule : UPDATE employes SET service='commercial', salaire=1872 WHERE id_employes=990; // Ici je modife le service et le salaire de l'employé qui a l'ID 990.

b) Supprimer avec DELETE

  • Supprimer l'enregistrement d'un employé avec DELETE : DELETE FROM employes WHERE id_employes=388;
    Attention, il est très facile de faire une erreur lors d'une suppression ou d'une maj // vérifiez bien la manipulation en faisant un SELECT * FROM table opur vérifier que vous n'avaez pas fait une mauvais manipulation.
  • Supprimer plusieurs enregistrements avec le code suivant : DELETE FROM employes WHERE service = 'informatique'; ici on supprime toutes les personnes du service informatique. // NE PAS LE FAIRE, exemple !

3 - Quelques exercices // Mettez la requête que vous faites dans une balise code à l'intérieur de la li

  • Afficher la profession de l'employé 547 : SELECT service FROM employes WHERE id_employes = 547;
  • Afficher la date d'embauche d'Amandine : SELECT date_embauche FROM employes WHERE prenom='Amandine';
  • Afficher le nombre de commerciaux : SELECT COUNT(*) AS nombre_commerciaux FROM employes WHERE service='commercial';
  • Afficher le coût des commerciaux sur une année : SELECT SUM(salaire*12) AS salaire_annuel_commercial FROM employes WHERE service = 'commercial';
  • Afficher le salaire moyen par service : SELECT service, AVG(salaire) AS salaire_moyen FROM employes GROUP BY service;
  • Afficher le nombre de recrutement sur l'année 2010 : SELECT date_embauche, COUNT(*) AS nb_recrutement_2010 FROM employes WHERE date_embauche BETWEEN '2010-01-01' AND '2010-12-31' GROUP BY date_embauche;
    SELECT COUNT(*) AS 'nbr de recrutements - 2010' FROM employes WHERE date_embauche LIKE '2010%';
    SELECT COUNT(*) AS 'nbr de recrutements - 2010' FROM employes WHERE date_embauche >= '2010-01-01' AND date_embauche <= '2010-12-31';
  • Augmenter le salaire pour chaque employé de 100€ : UPDATE employes SET salaire=salaire+100;
  • Afficher le nombre de service : SELECT COUNT(DISTINCT service) AS 'nombre de services' FROM employes;
  • Afficher les informations de l'employé du service commercial gagnant le salaire le plus élevé : SELECT * FROM employes WHERE salaire=(SELECT MAX(salaire) FROM employes WHERE service='commercial');
  • Afficher l'employé ayant été embauché en dernier : SELECT * FROM employes WHERE date_embauche IN(SELECT MAX(date_embauche) FROM employes);
